Customer Registry System
📌 Project Overview

The Customer Registry System is a Full Stack MERN application developed to manage customer information and customer service requests efficiently. It allows users to add, view, and delete customer records while storing all data in MongoDB.

The system helps organizations maintain a centralized customer database and track customer issues for better service management.

🚀 Features
Add new customer records
View all customer records
Delete customer records
Store customer data in MongoDB
Real-time frontend and backend integration
Simple and user-friendly interface
